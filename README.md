# Nanocode Compiler
Compilador educacional simplificado desenvolvido como projeto acadêmico para demonstrar na prática os conceitos fundamentais de Teoria da Computação e Compiladores.

# Conceitos Abordados
Hierarquia de Chomsky (Tipos 0–3);
Autômato Finito Determinístico (AFD);
Autômato de Pilha (PDA);
Gramáticas BNF;
Tabela de Símbolos;
Máquina de Turing;
